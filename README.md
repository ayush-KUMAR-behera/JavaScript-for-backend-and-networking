# JavaScript for Backend Engineers

A structured JavaScript learning journey focused on the concepts required for **Backend Development**, **Computer Networks**, and **Frontend–Backend Communication**.

## Why this repository?

As a Java Backend Developer, I wanted to understand how browsers interact with backend services instead of learning JavaScript only for frontend frameworks.

This repository documents my daily learning with hands-on coding and practice exercises.

## Learning Objectives

* Learn the JavaScript fundamentals required for backend engineers.
* Understand how browsers execute JavaScript.
* Learn DOM manipulation and browser events.
* Understand asynchronous JavaScript (Callbacks, Promises, Async/Await).
* Learn how `fetch()` communicates with backend APIs.
* Understand JSON, HTTP requests, headers, cookies, and JWT authentication.
* Build a strong foundation for Computer Networks by connecting JavaScript with real-world client-server communication.
